EcoBusca - Localizador de Pontos de Reciclagem

Este projeto é uma plataforma desenvolvida para incentivar a reciclagem, ajudando usuários a encontrar os ecopontos mais próximos. A interface foi construída com foco na experiência do usuário, utilizando HTML5 para a estruturação e CSS3 com Flexbox para criar um layout responsivo e de fácil navegação, que se adapta a diferentes dispositivos.
